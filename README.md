This is examples of Design Patterns in Crystal

- Abstract Factory Pattern  
- Builder Pattern  
- Lazy Initialization Pattern  
  
Structural patterns:
- Adapter Pattern  
- Bridge Pattern  
- Composite Pattern  
- Decorator Pattern  
- Facade Pattern  
- Flyweight Pattern  
- Proxy Pattern
